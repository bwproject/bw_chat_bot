# 🤖 BW Chat Bot

  <b>Telegram-бот для автоматизации общения, обработки обращений и работы с Telegram Client API.</b>
</p>

---

## 📌 О проекте

**BW Chat Bot** — это Telegram-бот на Python, созданный для автоматизации общения с пользователями.

Проект объединяет Telegram Bot API и Telegram Client API, позволяя принимать сообщения, управлять диалогами и отправлять автоматические ответы.

---

## 🚀 Возможности

### 💬 Работа с сообщениями

- Приём сообщений от пользователей
- Обработка обращений через Telegram
- Отправка ответов пользователям
- Создание пользовательских диалогов

### 👤 Telegram Client API

Используется **Telethon**:

- работа с Telegram аккаунтом
- управление пользовательской сессией
- подключение через Telegram API
- создание новых чатов

### ⚡ Автоответы

Поддерживается автоматическая отправка сообщений:

- настройка задержки ответа
- сообщение при отсутствии администратора
- собственный текст автоответа

Пример:

```
Сейчас мы не в сети.
Мы ответим вам позже.
```

---

## 🌐 Поддержка прокси

Бот поддерживает запуск через прокси-соединение.

Включение прокси:

```python
USE_PROXY = True
```

Отключение:

```python
USE_PROXY = False
```

## ⚙️ Установка

Клонирование проекта:

```bash
git clone https://github.com/bwproject/bw_chat_bot.git
cd bw_chat_bot
```

Установка зависимостей:

```bash
pip install -r requirements.txt
```

Создайте файл `.env`:

```env
BOT_TOKEN=ваш_токен_бота
ADMIN_ID=ваш_ID
API_ID=telegram_api_id
API_HASH=telegram_api_hash
AUTO_REPLY_DELAY=300
AUTO_REPLY_TEXT=Сейчас мы не в сети.
```

---

## ▶️ Запуск

```bash
python main.py
```

После запуска бот подключится к Telegram и начнёт обработку сообщений.

---

## 🔐 Безопасность

Не публикуйте:

- `.env`
- BOT_TOKEN
- API_HASH
- файл `session.session`

Рекомендуется использовать отдельный Telegram аккаунт для клиентской сессии.
